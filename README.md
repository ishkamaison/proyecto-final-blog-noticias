# 📰 Blog de Noticias Chile - Proyecto de Certificación

Este es el proyecto final para la certificación de **Desarrollo Full Stack JavaScript**. 
Es una aplicación completa que permite la gestión de noticias con un sistema de interacción social.

## 🚀 Enlaces del Proyecto
- **App Desplegada:** [LINK_DE_RENDER_O_GITHUB_PAGES]

## 🛠️ Tecnologías Utilizadas
- **Backend:** Node.js, Express.
- **Base de Datos:** PostgreSQL (Relacional).
- **Seguridad:** JWT (JSON Web Tokens) y Bcrypt para encriptación.
- **Frontend:** HTML5, CSS3, Bootstrap 5 y JavaScript Vanilla.

## 📋 Requisitos Cumplidos
- [x] Registro y Login de Usuarios.
- [x] Publicación de noticias (Máx 60 caracteres título / 4000 contenido).
- [x] Sistema de Likes y Dislikes con lista de votantes.
- [x] Ordenamiento cronológico ascendente y descendente.
- [x] Arquitectura MVC y rutas protegidas.

## ⚙️ Instalación Local
1. Clonar el repositorio.
2. Ejecutar `npm install`.
3. Configurar el archivo `.env` basado en `.env.example`.
4. Ejecutar los scripts de `documentos/tablas.sql` en PostgreSQL.
5. Iniciar con `npm run dev`.
